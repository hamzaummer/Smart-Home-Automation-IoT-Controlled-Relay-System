# =============================================================================
# HARDWARE REQUIREMENTS
# =============================================================================

# Essential Components:
# - Raspberry Pi Pico W (1x) - WiFi-enabled microcontroller
# - Single Channel Relay Module (1x) - 3.3V or 5V with optocoupler isolation
# - Jumper Wires (6x) - Male-to-male connecting wires
# - Micro USB Cable (1x) - For programming and power
# - Breadboard (1x) - For prototyping connections (optional)

# Electrical Load:
# - Any AC/DC device you want to control (bulb, fan, pump, etc.)
# - Ensure proper electrical safety for AC loads
# - Use appropriate relay rating for your load

# =============================================================================
# SOFTWARE REQUIREMENTS
# =============================================================================

# For Raspberry Pi Pico W:
# - MicroPython firmware (latest version for Pico W)
# - All required modules are built into MicroPython:
#   * machine (GPIO control)
#   * network (WiFi connectivity)
#   * socket (HTTP server)
#   * time (timing functions)
#   * json (JSON parsing)
#   * gc (garbage collection)
#   * hashlib (security functions)
#   * re (regular expressions)

# For Development/Testing on PC (optional):
# - Python 3.8 or newer
# - No external packages required (uses standard library only)

# Development Tools:
# - Thonny IDE (recommended) - thonny.org
# - Or any MicroPython-compatible IDE
# - Web browser for accessing the interface

# =============================================================================
# NETWORK REQUIREMENTS
# =============================================================================

# WiFi Network:
# - 2.4GHz WiFi network (Pico W limitation)
# - WPA2/WPA3 security (recommended)
# - Stable internet connection
# - Network access for the Pico W device

# Client Access:
# - Web browser on the same network
# - Modern browser with JavaScript support
# - Mobile device support (responsive design)

# =============================================================================
# SYSTEM REQUIREMENTS
# =============================================================================

# Memory:
# - Minimum 100KB free RAM on Pico W
# - 1MB flash storage for all files
# - Automatic memory management included

# Performance:
# - Designed for 24/7 operation
# - Handles up to 5 concurrent connections
# - 60 requests per minute rate limiting

# Security:
# - Change default password before deployment
# - Use secure WiFi network
# - Consider network isolation for IoT devices
